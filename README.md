# HowTo

```
$ git clone git@github.com:mashhadlug/next.git
$ cd next
$ virtualenv3 .env
$ source .env/bin/activate
$ pip install pelican
$ make devserver
```

You can open [localhost:8000](http://localhost:8000/).
Also To publish changes to github's branch simply run:

```
# .. make changes
$ make github
```
